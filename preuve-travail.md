# MEMO GIT / GITHUB



Add commit message 

```bash
$ git commit -m “Commit message” -a

-m = commit message is going to follow

-a = commit should include **a**ll files
```

Push to server

```bash

$ git push
```

Pushing to server (ex Github)

```bash
$ git remote add “remote name → origin” ssh://git-server

$ git remote -v
```
